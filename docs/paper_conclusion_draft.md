# 结论（论文撰写草稿）

## 结论

本文围绕功耗受限条件下的多传感器调度问题，构建了一套“共享真值环境—调度策略回放—状态估计—下游预测评估”的统一实验框架，并在同一高频风吹雪合成环境上系统比较了 `full_open`、`random`、`periodic`、`round_robin`、`info_priority` 与基于 DQN 的学习型调度器。与仅考察估计误差或缺失恢复质量的做法不同，本文将评价重点放在调度策略对下游预测任务的实际影响，即在节电约束下能否尽可能保持风吹雪目标变量 `snow_mass_flux_kg_m2_s` 的预测精度。

实验结果表明，调度策略对下游预测效果具有显著影响，而且“估计侧最优”并不等价于“预测侧最优”。在最新一轮完整实验（`full_schemeA_v4`）中，`round_robin` 在平均节电约 `45.7%` 的条件下，跨预测模型的平均 RMSE 相对 `full_open` 反而降低了约 `3.53%`；`info_priority` 在近似相同的节电比例下，平均 RMSE 仅增加约 `2.18%`。相比之下，`periodic` 与 `random` 虽然能够达到约 `65%` 的节电，但其平均 RMSE 分别增加约 `10.23%` 和 `12.17%`，说明过于激进的节电会直接损伤下游预测质量。上述结果说明，在当前任务设定下，具有覆盖保证或目标相关性偏好的规则策略，已经能够形成非常强的功耗—精度折中基线。

对于学习型调度器，本文的一个重要发现是：如果奖励函数主要优化估计不确定性，那么 DQN 很容易学到对下游预测并不友好的采样行为；而当奖励重新调整为以预测效果为主、功耗仅作为硬约束时，DQN 的行为会明显改善。与此前 estimator-driven 的版本相比，当前 forecast-aware 版本的 DQN 已经不再产生明显失真的数据分布，也不再在多数预测模型上出现灾难性退化。在 `full_schemeA_v4` 中，DQN 在平均节电约 `42.95%` 的条件下，跨模型平均 RMSE 相对 `full_open` 仅增加约 `0.83%`。这说明基于学习的调度器经过合理的目标设计后，已经能够在有限功率预算下较好地保持预测性能。

然而，实验同样表明，当前 DQN 尚未稳定超过最强规则基线。尽管 DQN 在部分模型上已经接近甚至超过 `full_open`，例如在 `PINN` 与 `Transformer` 上出现了相对较好的结果，但这种优势尚未在所有预测模型上稳定复现；与之相对，`round_robin` 与 `info_priority` 的表现更为稳健。这一现象说明，当前问题的难点并不只是“如何学习一个策略”，而在于如何让奖励函数、状态估计器与最终任务目标真正一致。换言之，当前学习型调度器已证明了可行性，但还没有充分证明其必要性或最优性。

本文还考察了在下游预测器中引入轻量物理约束的效果。实验将 `PINN` 作为新的评估模型加入，与 `MLP`、`LSTM`、`Transformer`、`Informer`、`TCN` 和 `Naive` 一并比较。结果显示，加入非负约束、阈值输运约束和事件加权后，`PINN` 可以在个别调度策略上取得优于 `full_open` 的单目标、单步长结果，但并未从根本上改变不同调度策略之间的总体排序。这说明：轻量物理约束能够提升预测器对已有信息的利用效率，但无法替代调度端对关键信息的保留；当上游采样决策已经造成信息损失时，仅在预测模型端施加约束不足以完全弥补。

综合来看，本文的主要结论可以概括为三点。第一，调度策略必须围绕最终预测任务而非单纯估计指标进行设计，否则容易出现“协方差更小但预测更差”的目标错配。第二，在当前风吹雪任务中，具有覆盖约束或目标相关性启发的规则调度策略是非常强的基线，任何学习型方法都必须在这一基线上证明自身价值。第三，学习型调度器在预测导向奖励下已表现出明显改善，说明基于强化学习的功耗受限感知仍然具有研究潜力，但其优势尚需通过更合理的状态表示、更一致的奖励构造以及更稳定的多随机种子实验进一步验证。

## 局限性与后续工作

本文仍存在若干限制。首先，当前状态估计器虽然已由简单随机游走近似改进为数据拟合的线性高斯代理模型，但其本质上仍是任务导向 surrogate，而非具有强物理解释力的风吹雪动力学模型。其次，本文的下游目标仍聚焦于单一目标变量 `snow_mass_flux_kg_m2_s`，这会天然强化 `info_priority` 这类目标相关性启发式的优势。再次，当前结论主要基于单轮完整实验与多模型横向对比，尚未通过多随机种子和统计显著性检验给出更强的置信结论。

基于上述局限，后续工作可沿三个方向展开：其一，进一步改进奖励函数，使其更直接地反映短期与中期预测收益，并减少估计侧 surrogate 对策略学习的支配；其二，引入更合理的物理或半物理状态模型，提升调度—估计—预测链条的一致性；其三，在现有框架上开展多随机种子、多目标变量和更丰富事件分析的系统实验，以更严格地评估学习型调度器是否能够在强规则基线之上取得稳定优势。
